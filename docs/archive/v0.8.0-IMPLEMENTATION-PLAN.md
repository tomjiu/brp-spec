**Status**: Draft

## 1. v0.8.0 Goal

v0.7.0 added test infrastructure. v0.8.0 is the last content version before v1.0:

- E2E tests in CI (xvfb, non-blocking)
- API documentation (protocol contract)
- README/USAGE/SECURITY polish
- Regression tests (backward compat)
- API freeze declaration

### Non-goals (deferred to v1.0)

| Item | Deferred to | Reason |
|------|:---:|--------|
| supportsAutoLink capability | v1.0 | Single version, no mixed-version use case |
| LibreWolf compat | v1.0 | Firefox fork, should be compat |
| Performance tests | v1.0+ | Not needed for stability |
